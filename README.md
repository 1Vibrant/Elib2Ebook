## Пример вызова сервиса
```
Elib2Ebook.exe --url https://litnet.com/ru/book/kniga-7-vladyka-magii-b364041 --format epub
```

## Где 
```
--url - ссылка на книгу
--format - формат для сохранения
```

## Полный список опций 

```
Elib2Ebook.exe --help
```
